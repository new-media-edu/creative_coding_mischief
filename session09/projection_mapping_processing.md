# Projection Mapping: Processing (Custom)

Use your Processing projection mapper if you want a lightweight, code-forward setup.

- Open your Processing mapper sketch.
- Enter calibration/edit mode.
- Drag control points to match the projection surface.
- Save/load your calibration file if your sketch supports presets.
- Run in presentation/fullscreen mode on the projector display.

This is usually the fastest path when you already have a custom Processing pipeline.
